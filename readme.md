# **Codents Web**

_Como instalar el proyecto [rama: prototypes-react]_
```
$> git clone https://github.com/Codents/web.git
$> cd web
$> npm install
```

_Arrancar el proyecto en modo desarrollo y con watcher
```
$> npm run watch
```

_Construir el proyecto en modo release /build_
```
$> npm run build
```
